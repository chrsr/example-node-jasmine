# Example Node Jasmine app
Node.js + NPM + Jasmine getting started environment.

## Install dependencies

1. Install NodeJS + NPM.

2. Once you have NodeJS + NPM run the following:

    npm i

## Run tests

    npm run test

## Run tests + app

    npm run start

## Run app (without tests)

    npm run start-notest

or

    node app.js
